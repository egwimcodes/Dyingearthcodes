Process

Follow Our Installation Process

Explore our step-by-step installation guide for a seamless setup experience.

  • Step 1

    Unpack and Get the Sensor Code

    Unpack the sensor package and locate the card featuring the QR code for the sensor.

  • Step 2

    Create an Account, Login and Register your Sensor

    Visit https://dyingearthcodes.onrender.com/register/ to create an account and login.

    Go to Menu -> Add Sensor -> Add Sensor and register your sensor

  • Step 3

    Install the Sensor

    Go to the plant you want to monitor and install the sensor on the plant
    4 - 7 inches above the ground.

  • Step 4

    Monitor Your Sensor

    Monitor your sensor with the Dying Earth Sensor App. Visit https://dyingearthcodes.onrender.com/dashboard/

    Watch your plant Grow healthily..........
